## Neural Network Implementation from Scratch in Numpy

### Artificial Neural Network
Simple implementation of Backpropagation and gradient descent algorithm: [diabetes](https://www.geeksforgeeks.org/machine-learning/sklearn-diabetes-dataset/) dataset for regression task.

### Convolution Neural Network
Convolution Neural Network implementation for digit classification from [MNIST](https://en.wikipedia.org/wiki/MNIST_database) dataset. 

### Recurrent Neural Network
RNN implemetation with BPTT and Adagrad optimizer in numpy for target text generation.
